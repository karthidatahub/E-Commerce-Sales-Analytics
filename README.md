# 🛒 E-Commerce Sales Analytics (SQL Project)

## 📌 Project Overview

This project is an E-Commerce Sales Analytics database built using MySQL.

The project demonstrates SQL skills by creating a relational database, inserting sample business data, and solving real-world business problems using SQL queries.

---

## 🛠 Technologies Used

* MySQL
* SQL
* VS Code

---

## 📂 Project Files

* **01_Create_Database.sql** – Database and table creation
* **02_Insert_Data.sql** – Sample data insertion
* **03_Business_Analysis.sql** – 40 business analysis SQL queries
